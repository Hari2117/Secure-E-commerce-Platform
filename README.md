# Secure E-Commerce System

A secure Spring Boot backend for an e-commerce platform with JWT authentication, OAuth2 login, multi-tenancy, and security auditing.

## Features
- JWT-based authentication
- Role-based authorization
- Google & GitHub OAuth2 login
- Multi-tenant architecture
- Security audit logging
- Token blacklist for logout
- Docker support

## Tech Stack
- Java
- Spring Boot
- Spring Security
- JWT
- OAuth2
- PostgreSQL
- Docker

## Project Structure
src/main/java/com/ecommerce/
- security/
- model/
- controller/
- service/
